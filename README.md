# Cat's Heart v2 | Standalone Romantic Sanctuary

A fully self-contained, high-fidelity romantic web application designed for "Batman" from "Catwoman". 

## Key Features
- **Standalone Architecture**: All romantic text, letters, and logic are embedded directly in `index.html`. No local server or JSON fetching is required.
- **Mysterious Intro**: A 8-line timed transition sequences that establishes a "heart alignment" before entering the sanctuary.
- **The Vault**: A password-protected (`04092025`) repository of 5 deep letters, fragments of a soul.
- **Open When Letters**: 7 mood-based letters (Angry, Sad, Overwhelmed, Happy, Distant, Anxious, Miss Me) providing expert-tuned emotional support.
- **Sensory Engagement**: Staggered animations, Sacramento font for emotional depth, and visual/vibrational haptic feedback on every interaction.

## Project Structure
- `index.html`: Main container with embedded `<script id="app-data">`.
- `app.js`: Core logic, reusable modal rendering, and delayed button binding.
- `style.css`: Romantic palette (`#3a0050`, `#dbaaac`) and responsive layout.
- `animations.css`: Specialized transitions for intro, modals, and micro-interactions.
- `assets/`: 
    - `images/`: Put your shared photos here.
    - `fonts/`: Dedicated space for local font files if offline usage is needed.
    - `sounds/`: Space for ambient tracks or notification pulses.

## How to Use
1. Simply open `index.html` in any modern web browser.
2. Wait for the **Alignment Process** to complete.
3. Use the password **04092025** to unlock the Vault.
